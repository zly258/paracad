# ParaCad

ParaCad is a browser-based, node-driven parametric modeling platform built with React and Three.js.

## Highlights

- Node graph workflow for sketch, solid, transform, boolean, and analysis.
- Real-time Three.js viewport with fast interaction.
- Expression node with math helpers and ternary support.
- Export support: `GLB` / `OBJ`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## 中文说明

ParaCad 是一个基于 React + Three.js 的浏览器端节点式参数化建模平台。

- 覆盖草图、实体、变换、布尔与分析节点。
- 提供实时三维预览与快速交互。
- 表达式节点支持三目运算与常用数学函数。
- 支持导出 `GLB` / `OBJ`。
